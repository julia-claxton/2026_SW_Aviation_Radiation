# **On the Role of Electron Precipitation in Excess Radiation Doses Measured at Aviation Altitudes**
#### Julia Claxton (she/they) || julia.claxton@colorado.edu

This repository contains the code and data necessary to replicate the analysis, results, and figures from the paper *"On the Role of Electron Precipitation in Excess Radiation Doses Measured at Aviation Altitudes"* submitted to *Space Weather*.

The manuscript for the paper may be found at `/paper/output/` in this repository. Source code for analysis and figure generation is in `/code/`. Figures in `.pdf`, `.png`, and `.svg` formats are located in `/paper/figures/`.

Please email me at julia.claxton@colorado.edu if you have any questions about this work!
